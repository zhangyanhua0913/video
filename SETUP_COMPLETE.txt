🚀 快速开始 (3 步)：

方式 1: 直接调用 Skill Handler
───────────────────────────────────────────────────────────
$ python skill_handler.py mix '{
    "output": "output.mp4",
    "clips": [
        {"path": "video1.mp4", "transition": "fade"},
        {"path": "video2.mp4"}
    ]
}'

方式 2: 使用 Python API
───────────────────────────────────────────────────────────
from skill_handler import VideoMixerSkillHandler

handler = VideoMixerSkillHandler()
result = handler.process('mix', {
    'output': 'output.mp4',
    'clips': [{'path': 'video1.mp4'}, {'path': 'video2.mp4'}]
})
print(result.to_json())

方式 3: 通过 OpenClaw 框架
───────────────────────────────────────────────────────────
# 安装 OpenClaw
pip install openclaw

# 注册 skill
openclaw skill register --name video-mixer --path /path/to/goEditVideo

# 调用 skill
openclaw skill call video-mixer mix \
  --output "output.mp4" \
  --clips '[{"path":"video1.mp4"}]'

───────────────────────────────────────────────────────────────

📋 Skill 支持的命令：

1. mix
   ├─ 功能: 混剪多个视频片段
   ├─ 参数: output*, clips*, resolution, textOverlays, 
   │       audioFadeIn, audioFadeOut, colorCorrection
   └─ 返回: {success, message, output, error}

2. thumbnail
   ├─ 功能: 生成视频缩略图
   ├─ 参数: videoPath*, output*, timestamp, width
   └─ 返回: {success, message, output, error}

3. saveConfig
   ├─ 功能: 保存混剪配置
   ├─ 参数: config*, output*
   └─ 返回: {success, message, output, error}

4. loadConfig
   ├─ 功能: 加载混剪配置
   ├─ 参数: configPath*
   └─ 返回: {success, message, output, error}

5. listTransitions
   ├─ 功能: 获取支持的转场效果
   ├─ 参数: 无
   └─ 返回: {success, message, output=[...]}

※ 带 * 的参数为必需参数

💡 实际使用示例：

示例 1: YouTube 视频（通过 OpenClaw）
─────────────────────────────────────
openclaw skill call video-mixer mix \
  --output "youtube.mp4" \
  --resolution '[1920, 1080]' \
  --clips '[
    {"path":"intro.mp4","transition":"fade","transitionDuration":1},
    {"path":"content.mp4","transition":"wipeleft","transitionDuration":1},
    {"path":"outro.mp4","transition":"fade","transitionDuration":1}
  ]' \
  --textOverlays '[
    {"text":"欢迎订阅","startTime":0,"duration":3,"fontsize":48},
    {"text":"点赞和订阅","startTime":8,"duration":2,"y":"bottom"}
  ]' \
  --audioFadeIn 1.0 \
  --audioFadeOut 1.5

示例 2: 社交媒体视频（通过 Python API）
──────────────────────────────────────
from skill_handler import VideoMixerSkillHandler

handler = VideoMixerSkillHandler()
result = handler.process('mix', {
    'output': 'tiktok.mp4',
    'resolution': [1080, 1920],  # 竖屏
    'clips': [
        {'path': 'clip1.mp4', 'transition': 'fade', 'transitionDuration': 0.5},
        {'path': 'clip2.mp4', 'transition': 'zoomin', 'transitionDuration': 0.5},
        {'path': 'clip3.mp4', 'transition': 'wipeleft', 'transitionDuration': 0.5}
    ],
    'textOverlays': [
        {'text': '快速混剪', 'startTime': 0, 'duration': 1},
        {'text': '点赞👍', 'startTime': 2, 'duration': 1}
    ],
    'audioFadeIn': 0.5,
    'audioFadeOut': 0.5
})

if result.success:
    print(f"视频已生成: {result.output}")
