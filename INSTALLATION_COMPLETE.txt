🚀 快速开始 (三选一)：

方式 1️⃣ : 直接调用 Skill Handler
─────────────────────────────────────
$ python skill_handler.py mix '{
    "output": "output.mp4",
    "clips": [{"path": "video1.mp4"}]
}'

方式 2️⃣ : Python API 调用
─────────────────────────────────────
from skill_handler import VideoMixerSkillHandler

handler = VideoMixerSkillHandler()
result = handler.process('mix', {
    'output': 'output.mp4',
    'clips': [{'path': 'video1.mp4'}]
})

方式 3️⃣ : OpenClaw 框架调用
─────────────────────────────────────
$ pip install openclaw
$ openclaw skill register --name video-mixer --path /path/to/goEditVideo
$ openclaw skill call video-mixer mix --output "output.mp4" --clips '[...]'

🔍 Skill 命令参考：

1. mix
   功能: 混剪多个视频片段
   参数: output* clips* resolution textOverlays audioFadeIn 
         audioFadeOut colorCorrection
   示例: openclaw skill call video-mixer mix --output "out.mp4" \
         --clips '[{"path":"v1.mp4"}]'

2. thumbnail
   功能: 生成视频缩略图
   参数: videoPath* output* timestamp width
   示例: openclaw skill call video-mixer thumbnail \
         --videoPath "video.mp4" --output "thumb.jpg"

3. listTransitions
   功能: 获取转场效果列表
   参数: 无

4. saveConfig
   功能: 保存混剪配置
   参数: config* output*

5. loadConfig
   功能: 加载混剪配置
   参数: configPath*

※ 带 * 的参数为必需参数
