🚀 快速开始 (3 步)：

1. 安装 FFmpeg
   Windows: choco install ffmpeg
   macOS:   brew install ffmpeg
   Linux:   sudo apt-get install ffmpeg

2. 运行示例
   python example_usage.py
   或
   python advanced_examples.py

3. 创建自己的视频
   from video_mixer import VideoMixer
   mixer = VideoMixer(output_path='my_video.mp4')
   mixer.add_clip('video1.mp4')
   mixer.add_clip('video2.mp4')
   mixer.generate()

💡 使用示例：

基础使用：
  mixer = VideoMixer()
  mixer.add_clip('video1.mp4')
  mixer.add_clip('video2.mp4')
  mixer.generate()

完整功能：
  mixer = VideoMixer(output_path='output.mp4')
  mixer.add_clip('v1.mp4', transition='fade', transition_duration=1.0)
  mixer.add_clip('v2.mp4', transition='wipeleft', transition_duration=1.5)
  mixer.add_text_overlay('标题', start_time=0, duration=3, fontsize=48)
  mixer.set_audio_fade(fade_in=1.0, fade_out=2.0)
  mixer.enable_color_correction(brightness=0.1, contrast=1.2)
  mixer.generate()
  mixer.generate_thumbnail('thumb.jpg', timestamp=5)
  mixer.save_config('config.json')
